# Emotion Detection Application using Watson NLP

## Overview
This project is a Flask-based web application that performs emotion detection on user-provided text using the Watson NLP Emotion Predict service.

The application analyzes input text and identifies five different emotions:

- Anger
- Disgust
- Fear
- Joy
- Sadness

It also determines the dominant emotion based on the highest confidence score.

## Features

- Accepts user text input
- Sends text data to Watson NLP Emotion Prediction API
- Processes JSON responses
- Displays emotion confidence scores
- Identifies the dominant emotion
- Handles invalid or blank input entries
- Provides a simple web interface using Flask

## Technologies Used

- Python
- Flask
- Watson NLP Library
- HTML
- JavaScript
- PyLint
